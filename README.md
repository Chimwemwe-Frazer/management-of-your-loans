# Loan Management System

A Django-based Loan Management System that allows users to apply for loans, view their loan history, and manage loan due dates securely.

This project was built for learning purposes and to practice real-world Django development concepts such as authentication, models, signals, and user-specific data handling.

---

## Features
- User authentication (login & logout)
- Loan application system
- Automatic loan due date calculation
- User-specific loan history
- Secure access using Django authentication
- Borrower profile created automatically using Django signals

---

## Tech Stack
- Python
- Django
- SQLite (development database)
- HTML / CSS (Django templates)
